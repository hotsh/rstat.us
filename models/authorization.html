An Authorization represents a connection to someone’s social media profile. For example, a User might have two Authorizations, one for their Twitter and one for their Facebook.

 
class Authorization
  include MongoMapper::Document

  belongs_to :user

  key :uid, Integer, :required => true
  key :provider, String, :required => true
  key :oauth_token, String
  key :oauth_secret, String
  key :nickname

  validates_uniqueness_of :uid, :scope => :provider
